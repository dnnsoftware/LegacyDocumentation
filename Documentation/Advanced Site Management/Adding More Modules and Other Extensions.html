Adding More Modules and Other Extensions

Modules can be installed and allocated to sites using the Host > Extensions page. The Host > Extensions page allows SuperUsers to install modules and allocate them to one or more sites. The Host > Extensions page manages extensions of all types including authentication systems, skins, containers, language packs and provider.
