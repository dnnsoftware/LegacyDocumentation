Setting Modules as Translated

How to set one or one or more modules as translated.

  1. Select Page Localization from the Edit Page menu of the ControlBar - OR - Navigate to Admin > Advanced Settings > Languages - OR - Go to a Languages module.
  2. Go to the Page Modules section of the required language.
  3. In the Translated? column, beside each module that is translated, or click the Translated? button itself to set all translatable modules as translated.

  1. Click the Update Localization button.
