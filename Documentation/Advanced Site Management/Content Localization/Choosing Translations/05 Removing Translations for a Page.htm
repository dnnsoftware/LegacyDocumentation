Removing Translations for a Page

How to remove translations for a language from a page using the Languages module. This allows you to translate some pages into all enabled languages and other pages only translated into some languages which is useful when different site information is targeted towards different cultures and languages. Removed translations can also be added back to the page, allowing you to restart a translation from scratch.

  1. Navigate to Admin > Advanced Settings > Languages - OR - Go to a Languages module.
  2. Select the Content Localization tab.
  3. At Select a Page, find or select the required page.
  4. Click the Delete button beside each translation to be deleted.

  1. This displays the message "You are about to permanently remove the 'fr-CA' translation of this page. Are you sure about this?"
  2. Click the Yes button to confirm.
