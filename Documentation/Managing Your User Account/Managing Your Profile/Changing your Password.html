Changing your Password

How to update the password associated with your user profile. Important. If you are updating the default SuperUser or Administrator login details for the first time, begin this tutorial at Step 5.

  1. Login to the site. See "Logging into a Site"
  2. Click on your [Display Name] link (typically located in the top right corner of the site) - OR - Navigate to a View Profile module (also called the My Profile module). This displays the User Profile page.
  3. Click the Edit Profile button. This displays the Manage Profile page.
  4. Select the Manage Account tab.
  5. Expand the Manage Password section.
  6. In the Change Password section, update the following fields:
    1. In the Current Password text box, enter your current password.
    2. In the New Password text box, enter your new password.
    3. In the Confirm Password text box, re-enter your new password.

  1. Click the Change Password button.
