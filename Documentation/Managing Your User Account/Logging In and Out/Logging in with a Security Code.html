Logging in with a Security Code

How to login to a site when the CAPTCHA security code is required. This is a unique code that is generated as a picture each time you login.

  1. Click the Login link (typically located in the top right corner of each page) to open the User Login pop-up window - OR - Navigate to an Account Login module.

  1. In the Username text box, enter your user name.
  2. In the Password text box, enter your password.
  3. In the Security Code text box, enter the code displayed as a picture.
  4. Click the Login button.
