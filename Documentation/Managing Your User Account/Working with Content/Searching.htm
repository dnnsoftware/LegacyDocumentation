Searching the Site

How to search for content including files other than site pages using the default search box that is typically displayed in the top right corner of each page.

Tip: Searches are locale aware, meaning they will only search in the locale/language that you are currently using.

  1. Enter your search criteria in the search box displayed in the top right corner of any page. The top five results are displayed as you type.


Search Results displaying the first five page matches


Search Results displaying the first five matches including indexed file types and file content

  1. Select from these options:
    • Click on one of the top five results to view that result. If the result is a site pages you will be taken to the page. If the result a file such as an image or a document, the file will be opened or available for opening directly from the search results.
    • Click the See More Results link or the Search  button to go to Search Results page where you can view additional results and configure additional advanced options. See "Performing an Advanced Search"
