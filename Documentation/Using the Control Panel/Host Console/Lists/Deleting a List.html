Deleting a List

How to delete a list from the Host Lists page. Note: Only user generated lists can be deleted.

  1. Navigate to Host > Lists.
  2. In the Lists section, navigate to and select the title of the required list. This displays details of this list and its related list entries.
  3. Click the Delete List button. This displays the message "Are You Sure You Wish To Delete This Item?"
  4. Click the Yes button confirm.
