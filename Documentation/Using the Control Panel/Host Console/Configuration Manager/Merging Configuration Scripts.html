Merging Configuration Scripts

How to merge configuration scripts using the Configuration Manager module.

  1. Navigate to Host >  Configuration Manager.
  2. Expand the Configuration Files section.
  3. At Configuration File, select the file to be modified from the drop down list. This loads the selected file into the File text box.

  1. Expand the Configuration Merge Scripts section.
  2. At Merge Script, select from these option:
    • To upload a script file:
      1. Click the Browse... button.
      2. Select the required file.
      3. Click the Upload link. This loads the selected file into the text window below.
    • or paste your script into the Scripts text box.
  3. Click the Execute Merge link. Review any message which is displayed and then click OK to continue.

  1. Optional. To view the successful merge, search for a part of the merged script in the Configuration Files section.
