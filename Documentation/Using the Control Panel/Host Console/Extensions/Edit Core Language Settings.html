Edit Core Language Settings

How to edit the language package setting using the Extensions module.

  1. Navigate to Host >  Extensions.
  2. Expand either the Core Language Pack or Extension Language Pack section as required. This opens the Edit Extension page.
  3. Click the Edit button beside the required language. This opens the Edit Extension page.
  4. Expand the Language Pack Settings section.
  5. At Language, select the associated language.
  6. Click the Update Extension link.
