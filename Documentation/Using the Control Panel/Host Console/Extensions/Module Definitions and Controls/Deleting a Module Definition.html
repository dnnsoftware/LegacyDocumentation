Deleting a Module Definition

How to delete a definition associated with a module.

  1. Navigate to Host > Extensions.
  2. Select the Installed Extensions tab.
  3. Expand the Modules section.
  4. Click the Edit button beside the required module.
  5. Expand the Module Definitions section.
  6. At Select Definition, select the definition to be deleted from the drop down list.
  7. Click the Delete Definition link.
