Editing a Module Definition

How to edit a definition associated with a module.

  1. Navigate to Host > Extensions.
  2. Select the Installed Extensions tab.
  3. Expand the Modules section.
  4. Click the Edit button beside the required module.
  5. Expand the Module Definitions section.
  6. At Select Definition, select the definition to be edited from the drop down list.
  7. In the Friendly Name text box, edit the friendly name for this definition.
  8. In the Default Cache Time text box, edit the default cache time for this module. A default cache time of "-1" indicates that the module does not support output caching.
  9. Click the Update Definition link.
