Editing Module Package Settings

How to edit the package settings of a module such as friendly name, description, version, license, release notes, owner, organization name and contact details.

  1. Navigate to Host >  Extensions.
  2. Select the Installed Extensions tab.
  3. Expand the Modules section.
  4. Click the Edit button beside the required module.
  5. Go to the Package Settings section and edit one or more of these fields as required:
    1. At Name, the module name is displayed.
    2. In the Friendly Name text box, edit the friendly name of this module. This is the name the module is commonly known as and is the default module title.
    3. In the Description text box, edit the module description.
    4. At Version, edit the version number.
    5. In the License text box, edit the module license.
    6. In the Release Notes text box, edit the module release notes.
    7. In the Owner text box, edit the owner of this module.
    8. In the Organization text box, edit the name of the organization that created this module.
    9. In the Url text box, edit the URL of the owner/organization who created this module.
    10. In the Email Address text box, edit the email address of the owner/organization that created this module.
  6. Click the Update Module link.
