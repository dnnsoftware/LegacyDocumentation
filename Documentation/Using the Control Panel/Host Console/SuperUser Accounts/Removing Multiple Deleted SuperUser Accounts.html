Removing Multiple Deleted SuperUser Accounts

How to remove multiple SuperUser accounts which have been "soft" deleted from the SuperUser Accounts module. This action will permanently remove all information related to these accounts from your site's database.

  1. Navigate to Host > Advanced Settings > SuperUser Accounts.
  2. Click the Remove Deleted Users button. This displays the message "Are You Sure You Wish To Remove These Items?"
  3. Click the OK button to confirm.
