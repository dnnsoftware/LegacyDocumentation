Removing a Deleted SuperUser Account

How to remove a "soft" deleted SuperUser account using SuperUser User Accounts module. This removes all information related to this account from the database.

  1. Navigate to Host > Advanced Settings > SuperUser Accounts.
  2. Find the user to be deleted using a filter or by searching.
  3. Click the Remove button beside their record. This displays the message "Are You Sure You Wish To Remove This Item?"
  4. Click the OK button to confirm.
