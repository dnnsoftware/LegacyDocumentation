UnAuthorizing a SuperUser

How to unauthorize a SuperUser's account using the SuperUser Accounts module. Unauthorized SuperUser are unable to login to the site.

  1. Navigate to Host > Advanced Settings > SuperUser Accounts.
  2. Find the user to be unauthorized using a filter or by searching.
  3. Click the Edit button beside their record. This opens the Edit User Accounts page.
  4. Select the Manage User Credentials tab.
  5. Click the UnAuthorize User link. This displays the message "User Successfully UnAuthorized" and sets the Authorized field to False.
