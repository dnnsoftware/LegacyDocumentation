Deleting a SuperUser Account

How to "soft" delete a SuperUser account from a site using the SuperUser Accounts module. Information about this deleted account can still be viewed using this module.

  1. Navigate to Host > Advanced Settings > SuperUser Accounts.
  2. Find the SuperUser to be deleted using a filter or by searching.
  3. Click the Delete button beside their record. This displays the message "Are You Sure You Wish To Delete This Item?"
  4. Click the OK button to confirm.
