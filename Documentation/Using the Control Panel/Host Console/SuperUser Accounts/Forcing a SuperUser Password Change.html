Forcing a SuperUser Password Change

How to force a SuperUser to change their password next time they login to the site using the SuperUser Accounts module.

  1. Navigate to Host > Advanced Settings > SuperUser Accounts.
  2. Find the required user using a filter or by searching.
  3. Click the Edit button beside their user account.
  4. Select the Manage User Credentials tab.
  5. Click the Force Password Change link. This removes the Force Password Change link, sets the Update Password field to True and displays the "User must update password on next login" message.
