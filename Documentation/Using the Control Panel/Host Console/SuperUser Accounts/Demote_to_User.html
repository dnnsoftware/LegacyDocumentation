Demoting a SuperUser

How to demote a SuperUser to regular user. This removes all SuperUser privileges and moves the account to the Admin > User Accounts page.

  1. Navigate to Admin >  User Accounts - OR - Go to a User Accounts module.
  2. Find the required user using a filter or by searching.
  3. Click the Edit  button beside their user account.
  4. Select the Manage User Credentials tab.
  5. Click the Make Regular User button.
