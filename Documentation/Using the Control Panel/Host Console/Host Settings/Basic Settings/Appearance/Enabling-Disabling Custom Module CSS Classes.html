Enabling/Disabling Custom Module CSS Classes

How to enable a custom CSS class based on the module definition to be rendered. This can be used be skin designers to apply specific CSS styles to specific module types.

  1. Navigate to Host > Host Settings.
  2. Select the Basic Settings tab.
  3. Expand the Appearance section.
  4. At Use Custom Module Css Class?, select from these options:
    • Mark  the check box to enable custom CSS classes.
    • Unmark  the check box to disable custom CSS classes.

  1. Click the Update button.
