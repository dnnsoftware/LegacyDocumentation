Setting the Host Edit Skin and Container

How to set the default skin and container which are applied to the editing pages for all sites within this DNN installation.

  1. Navigate to Host > Host Settings.
  2. Select the Basic Settings tab.
  3. Expand the Appearance section.
  4. At Edit Skin, select the skin for all Host pages from the drop down list.
  5. At Edit Container, select the container for all Host pages from the drop down list.

  1. Optional. Click the Preview Edit Skin and Container button to preview the selected skin and container in a new Web browser.
  2. Click the Update button.
