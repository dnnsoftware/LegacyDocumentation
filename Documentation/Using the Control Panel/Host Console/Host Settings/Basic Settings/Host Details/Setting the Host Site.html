Setting the Host Site

By default, DNN loads the first site created as the default site. Here you can modify the site that is loaded by default. The Host site is also the site associated with the Host skin object. See "Setting the Host Skin and Container"

  1. Navigate to Host > Host Settings.
  2. Select the Basic Settings tab.
  3. Expand the Host Details section.
  4. At Host Website, select the host site from the drop down list.

  1. Click the Update button.
