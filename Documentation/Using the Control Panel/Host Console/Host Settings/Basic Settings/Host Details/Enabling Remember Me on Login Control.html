Enabling Remember Me on Login Control

How to enable the Remember Login check box on the login control (Account Login module). This setting controls all sites within this DNN installation. This field uses cookies that persist over multiple visits.

  1. Navigate to Host > Host Settings.
  2. Select the Basic Settings tab.
  3. Expand the Host Details section.
  4. At Enable Remember me on login controls?, select from the following options:
    • to enable the Remember Login check box.
    • to remove the Remember Me control.

  1. Click the Update button.


The Remember Login field enabled on the Login Control
