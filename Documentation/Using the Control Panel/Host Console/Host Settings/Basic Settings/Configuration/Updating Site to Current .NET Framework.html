Updating Site to Current .NET Framework

How to update your DNN installation to the current .NET Framework. This updates the Web.config file with the .NET 3.5 Framework. This framework may be required for some custom modules.

  1. Navigate to Host > Host Settings.
  2. Select the Basic Settings tab.
  3. Expand the Configuration section.
  4. At .NET Framework, the current framework details are displayed. If a more recent version is available the Update Site to current Framework link is displayed.
  5. Click the  Update Site to current Framework link.
