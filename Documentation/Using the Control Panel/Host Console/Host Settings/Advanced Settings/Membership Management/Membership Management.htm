Membership Management

How to configure the Host Settings for Membership Management. The settings are used to control passwords and to control whether Login IP address checking is enabled.

  1. Navigate to Host > Host Settings.
  2. Select the Advanced Settings tab.
  3. Expand the Membership Management section.
  4. At Reset Link Timeout (minutes) you can override the default time of 60 minutes to your own setting. When the time elapses the password reset link sent out will no longer be valid.
  5. At Enable Password History, to maintain a list of recently used passwords and prevent users from re-using their old passwords.
    1. If password history is enabled, enter the Number of passwords to store in the text box. The default is 5.
  6. At Enable password banned list, if you do not wish to check entered passwords against a list of common passwords.
  7. At Enable password strength check, if you do not want to display the password strength meter on the registration form.
  8. At Enable IP address checking, if you wish to setup Login IP Filters. See "Login Filters" on page 1

  1. Click the Update button.
