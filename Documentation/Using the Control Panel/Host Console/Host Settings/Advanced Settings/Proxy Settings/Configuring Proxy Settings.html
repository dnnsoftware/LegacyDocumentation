Configuring Proxy Settings

This setting may be required by your hosting provider to enable certain modules such as the RSS News Feed to process outgoing web requests.

  1. Navigate to Host > Host Settings.
  2. Select the Advanced Settings tab.
  3. Expand the Proxy Settings section.
  4. In the Proxy Server text box, enter the proxy server IP address. E.g. 192.1.2.102. Enter the domain or IP address if you are using a proxy server.
  5. In the Proxy Port text box, enter the proxy port address. E.g. 8021
  6. In the Proxy Username text box, enter the username provided by your hosting provider.
  7. In the Proxy Password text box, enter the password provided by your hosting provider.

  1. Click the Update button.
