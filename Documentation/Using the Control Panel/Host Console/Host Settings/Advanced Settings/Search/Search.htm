Configuring Search Admin Word Length

How to set the maximum and minimum words length to be included in searches by configuring the Host Setting.

  1. Navigate to Host > Host Settings.
  2. Select the Advanced Settings tab.
  3. Expand the Search section.
  4. In the Minimum Word Length text box, enter the minimum length word to search for as a numeric value. Words that are shorter than this value will be ignored during searches. The default setting is 3.
  5. In the Maximum Word Length text box, enter the maximum length of words that will be searched for as a numeric value. Words that are longer than this value will be ignored during searches. The default setting is 255.

  1. Click the Update button.
