Managing Allowable File Extensions

How to add, edit and remove the file extensions that are allowed to be uploaded to this DNN installation by site Administrators and other authorized users. The following file extensions are permitted by default: swf, jpg, jpeg, jpe, gif, bmp, png, svg, ttf, eot, woff, doc, docx, xls, xlsx, ppt, pptx, pdf, txt, xml, xsl, xsd, css, zip, template, htmtemplate, ico, avi, mpg, mpeg, mp3, wmv, mov and wav.

  1. Navigate to Host > Host Settings.
  2. Select the Other Settings tab.
  3. In Allowable File Extensions window, perform one of the following:
    • To add a file extension: Enter the extension preceded by a comma ( , ). E.g. ,fla,wma
    • To remove a file extension: Remove the three letter extension and the preceding comma ( , ).
