Allowing/Disallowing Content Localization

How to allow or disallow content localization to be created for sites within this DNN installation. Allowing content localization permits Administrators to enable and configure content localization on their site.

  1. Navigate to Host > Host Settings.
  2. Select the Other Settings tab.
  3. At Allow Content Localization?, select from the following options:
    • Mark  the check box to allow content localization.
    • Unmark  the check box to disable content localization and return the site to the default "en-us" (US English) setting. This is the default setting.

  1. Click the Update button. The "Enable Localized Content" button is now displayed on the Admin > Languages module, allowing Administrators to enable localized content as required.
