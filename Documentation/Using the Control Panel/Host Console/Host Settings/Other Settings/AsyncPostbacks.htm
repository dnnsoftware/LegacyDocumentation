Setting the Timeout for Asynchronous Postbacks

How to set the number of seconds that must expire with no response from the server before asynchronous postbacks time out during activities such as content staging and exporting/importing large sites.

  1. Navigate to Host > Host Settings.
  2. Select the Other Settings tab.
  3. In Async postbacks time out text box, enter the number of second. The value must be from 90-9999 seconds. The default setting is 90.
