Deleting User Entered Tags

How to delete tags which have been created by users.

  1. Navigate to Admin > Advanced Settings > Taxonomy - OR - Go to a Taxonomy Manager module.
  2. Click the Edit link beside the pre-existing entry named "Tags". This opens the Edit Vocabulary page.

  1. In the Terms list, select the tag to be edited. This displays the Edit Term section.
  2. Click the Delete link.
  3. Repeat Steps 3-4 to delete additional tags.
