Viewing Vocabularies List

Users who are authorized to view the Taxonomy Manager module are able to view the list of vocabularies which have been created. The Taxonomy Manager module displays the following information:

  • Name: The Vocabulary name
  • Description: The description given to the vocabulary
  • Type: Whether the vocabulary is a simple or hierarchical list
  • Scope: Whether the vocabulary is available to all sites in this DNN application (i.e. Application) or only this site (i.e. Site).
