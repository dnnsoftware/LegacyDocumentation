Viewing User Entered Tags

How to view the tags which have been created by users. SuperUsers are able to edit and delete tags.

  1. Navigate to Admin > Advanced Settings > Taxonomy - OR - Go to a Taxonomy Manager module.
  2. Click the Edit link beside the pre-existing entry named "Tags". This opens the Edit Vocabulary page where all of the tags are listed in the Terms list.

  1. In the Terms list, click on a tag. This displays the description in the Edit Term section.
  2. Repeat Step 2 to view descriptions for other tags.
  3. Click the Cancel button to return to the module.
