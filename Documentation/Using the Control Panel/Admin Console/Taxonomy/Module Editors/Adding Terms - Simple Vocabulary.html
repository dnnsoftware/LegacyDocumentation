Adding Terms (Simple Vocabulary)

How to add terms to a simple vocabulary using the Taxonomy Manager module.

  1. Navigate to Admin > Advanced Settings > Taxonomy - OR - Go to a Taxonomy Manager module.
  2. Click the Edit link beside the vocabulary for this term. This opens the Edit Vocabulary page.

  1. Click the Add New button. This displays the Create New Term section.
  2. In the Create New Term section, complete these fields:
    1. In the Name text box, enter a name for this term.
    2. In the Description text box, enter a description for this term.
  3. Click the Save button. The new term is now displayed in the Terms hierarchical list.
  4. Repeat Steps 3-5 to add additional terms.
