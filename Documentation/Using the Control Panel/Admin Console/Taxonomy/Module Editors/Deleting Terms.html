Deleting Terms

How to delete one or more term associated with either a simple or hierarchical vocabulary using the Taxonomy Manager module.

  1. Navigate to Admin > Advanced Settings > Taxonomy - OR - Go to a Taxonomy Manager module.
  2. Click the Edit link beside the vocabulary which the term is associated with. This opens the Edit Vocabulary page.
  3. In the Terms list, select the term to be edited. This displays the Edit Term section.
  4. Click the Delete link located below the Edit Term section.
  5. Repeat Steps 3-4 to delete additional terms.
  6. Click the Cancel button to return to the module.
