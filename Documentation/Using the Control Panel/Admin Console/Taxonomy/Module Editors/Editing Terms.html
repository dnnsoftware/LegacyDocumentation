Editing Terms

How to edit one or more term associated with either a simple or hierarchical vocabulary using the Taxonomy Manager module.

  1. Navigate to Admin > Advanced Settings > Taxonomy - OR - Go to a Taxonomy Manager module.
  2. Click the Edit link beside the vocabulary which the term is associated with. This opens the Edit Vocabulary page.
  3. In the Terms list, select the term to be edited. This displays the Edit Term section.
  4. In the Edit Term section, edit one or more fields:
    1. In the Name text box, edit the name of this term.
    2. In the Description text box, edit the description of this term.
    3. At Parent Term, select the parent term for this term. Note: This option is only displayed on Hierarchical Vocabularies.
  5. Click the Update button.
  6. Repeat Steps 3-5 to edit additional terms.
  7. Click the Cancel button to return to the module.
