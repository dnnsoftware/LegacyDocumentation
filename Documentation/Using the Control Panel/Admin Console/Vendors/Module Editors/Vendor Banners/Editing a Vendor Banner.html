Editing a Vendor Banner

How to edit the properties of a banner using the Vendors module.

  1. Locate the required vendor account by selecting a filter or doing a search.
  2. Click the Edit button beside the required vendor account. This opens the Edit Vendors page.
  3. Expand the Banner Advertising section. This displays all banners associated with this vendor.
  4. Click the Edit button beside the banner to be edited.

  1. Edit the required fields.
  2. Click the Update button.
