Editing an Affiliate Referral Account

How to edit the details of an affiliate referral account using the Vendors module.

  1. Locate the required vendor account by selecting a filter or by doing a search.
  2. Click the Edit button beside the required vendor account. This opens the Edit Vendors page.
  3. Expand the Affiliate Referrals section. This displays any affiliate referral accounts associated with this vendor.
  4. Click the Edit button beside the required record.
  5. Edit fields as required.
  6. Click the Update link.
