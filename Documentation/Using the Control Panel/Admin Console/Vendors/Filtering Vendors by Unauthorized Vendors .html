Filtering Vendors by Unauthorized Vendors

How to view all vendor accounts that have been set as unauthorized on the Vendors module. Note: The Unauthorized link only displays when there are one or more unauthorized vendors.

  1. Navigate to Admin > Advanced Settings >  Vendors.
  2. Click the Unauthorized link. This displays the first ten (10) matching records listed alphabetically by name. If there are more than ten matching records, then the Pager Control is displayed allowing you to access the other records.
