Searching Vendors by Email

How to search for a vendor account by email address on the Vendors module. This search returns exact matches only.

  1. Navigate to Admin > Advanced Settings >  Vendors.
  2. In the Search text box, enter the vendor's full email address.
  3. Select Email from the drop down list.
  4. Click the Search button. This displays the first ten (10) matching records listed alphabetically by name. If there are more than ten matching records, then the Pager Control is displayed allowing you to access the other records.

Tip: You must enter the full and exact email address.
