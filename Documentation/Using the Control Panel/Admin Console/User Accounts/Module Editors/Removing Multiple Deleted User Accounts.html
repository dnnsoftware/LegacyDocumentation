Removing Multiple Deleted User Accounts

How to remove multiple user accounts which have been "soft" deleted from the User Accounts module. This action will permanently remove all information related to these accounts from your site's database.

  1. Navigate to Admin >  User Accounts - OR - Go to a User Accounts module.
  2. Click the Remove Deleted Users link. This displays the message "Are You Sure You Wish To Remove These Items?"
  3. Click the Yes button to confirm.
