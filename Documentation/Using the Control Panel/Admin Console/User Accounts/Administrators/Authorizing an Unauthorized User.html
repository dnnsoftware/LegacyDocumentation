Authorizing an Unauthorized User

How to authorize an unauthorized user account using the User Accounts module.

  1. Navigate to Admin > User Accounts - OR - Go to a User Accounts module.
  2. Click the Unauthorized link to display only unauthorized accounts and find the required account.
  3. Click the Edit button beside their record.
  4. Click the Authorize User link. This displays the message "User Successfully Authorized" and sets the Authorized field to True.
