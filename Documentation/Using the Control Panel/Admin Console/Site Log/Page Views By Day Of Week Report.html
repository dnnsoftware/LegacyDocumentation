Page Views By Day Of Week Report

The Page Views By Day Of Week report displays a summary list of the number of visitors and users who viewed the site during the selected day range.

Report Fields:

  • Week Day: The day of the week
  • Views: Number of views for the day
  • Visitors: Number of all visitors for the day
  • Users: Number of registered user visits for the day
