Enabling the Site Log

The Site Log module is disabled by default and must be enabled and configured by a SuperUser before it can be used to generate reports. The disabled Site Log module displays the yellow warning message "Your Hosting Provider Has Disabled the Site Log History Feature For Your Portal".

Here's how to enable and configure the Site Log:

  1. Enable the Site Log and configure the default settings to be used for all sites, See "Configuring Site Log Settings for all Sites"
  2. Configure the Site Log history for the individual site, See "Editing Host Settings for a Site"
