Viewing a Site Report

How to view a site report using the Site Log module.

  1. Navigate to Admin > Advanced Settings >  Site Log - OR - Navigate to a Site Log module.
  2. At Report Type, select the required report. You can now skip to Step 5 if you want to view a report for last seven days including today.
  3. Optional. At Start Date, click the Calendar button and select the start date for the report.
  4. Optional. At End Date, click the Calendar button and select the end date for the report.
  5. Click the Display button to view the report results. If there aren't any matching results for the selected report and date range, then the "No records were found" message is displayed at the top of the module.
