Editing the Stylesheet

How to edit and save changes to the default stylesheet. This doesn't affect any styles that are part of a skin package uploaded to this site.

  1. Navigate to Admin > Site Settings.
  2. Select the Stylesheet Editor tab.
  3. In the multi-line text box, edit the current stylesheet.
  4. Click the Save Style Sheet link.
  5. Hold down the Ctrl button and press the F5 button - OR - Click the Refresh button on your Web browser to view the changes.
