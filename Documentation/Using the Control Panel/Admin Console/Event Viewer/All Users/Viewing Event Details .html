Viewing Event Details

How to display a detailed report of a logged event using the Log Viewer module.

  1. Navigate to Admin > Event Viewer - OR - Navigate to a Log Viewer module.
  2. Expand the Viewer section.
  3. Optional. At Type, select the required event type to filter records.
  4. Click on the colored bar of a record to view a detailed report.
  5. Optional. Click the record a second time to close the detailed report.
