Retrieve Password Instructions

The Retrieve Password instructions are displayed to a user when a question and answer is required to modify or retrieve their password. To locate this resource file, use the Find option on your web browser and search for MESSAGE_RETRIEVEPASSWORD_INSTRUCTIONS.Text .

Here is the default text of this site message in the DNN core language pack:

In order to retrieve your password, you will need to provide your user name, and the answer to the Password Question.
