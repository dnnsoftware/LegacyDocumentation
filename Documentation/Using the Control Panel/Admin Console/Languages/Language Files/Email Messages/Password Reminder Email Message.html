Password Reminder Email Message

The Password Reminder email message is sent to a user when they request a password reminder. To locate this resource file, use the Find option on your web browser and search for EMAIL_PASSWORD_REMINDER_SUBJECT.Text and EMAIL_PASSWORD_REMINDER_BODY.Text.

Here is the subject title and message for this email as provided in the default DNN core language pack:

[Portal:PortalName] Password Reminder

Dear [User:DisplayName],

You have requested a Password Reminder from [Portal:PortalName].

Please login using the following information:

Portal Website Address: [Portal:URL]
Username: [Membership:Username]
Password: [Membership:Password]

Sincerely,
[Portal:PortalName]

*Note: If you did not request a Password Reminder, please disregard this Message.
