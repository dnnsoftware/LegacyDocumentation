User Registration Private Email Message

The User Registration Private email message is used when site registration is set as Private. It is sent to the new user when they register for a user account. To locate this resource file, use the Find option on your web browser and search for EMAIL_USER_UPDATED_OWN_PASSWORD_SUBJECT.Text and EMAIL_USER_UPDATED_OWN_PASSWORD_TEXT.Text.

Here is the subject title and message for this email as provided in the default DNN core language pack:

[Portal:PortalName] Password Updated

Dear [User:DisplayName],

You have changed your Password on [Portal:PortalName].

Please login using the following information:

Website Address: [Portal:URL]

Username: [User:Username]

Sincerely,

[Portal:PortalName]
