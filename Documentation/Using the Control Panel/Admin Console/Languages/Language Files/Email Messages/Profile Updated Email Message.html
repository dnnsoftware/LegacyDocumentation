Profile Updated Email Message

The Profile Updated email message is sent to a user when they update their profile. To locate this resource file, use the Find option on your web browser and search for EMAIL_PROFILE_UPDATED_SUBJECT.Text and EMAIL_PROFILE_UPDATED_BODY.Text.

Here is the subject title and message for this email as provided in the default DNN core language pack:

[Portal:PortalName] Profile Update Notice

Dear [User:DisplayName],

Your profile on [Portal:PortalName] has been successfully updated.

Sincerely,
[Portal:PortalName]
