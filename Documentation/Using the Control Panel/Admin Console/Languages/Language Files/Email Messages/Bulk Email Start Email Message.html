Bulk Email Start Confirmation Email Message

The Bulk Email Start Confirmation email message is sent to the primary Administrator when a bulk email (newsletter) mail out commences. To locate this resource file, use the Find option on your web browser and search for EMAIL_BulkMailConf_Text_Body.Text and EMAIL_BulkMailConf_Subject.Text.

Here is the subject title and message for this email as provided in the default DNN core language pack:

Bulkmail '{0}' submitted

User {1} submitted Bulkmail '{0}' to {2} recipients. Please review the sendout report for failed transmissions and the senders inbox for returns.
