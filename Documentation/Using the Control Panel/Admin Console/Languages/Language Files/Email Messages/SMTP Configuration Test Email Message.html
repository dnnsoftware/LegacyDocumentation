SMTP Configuration Test Email Message

The SMTP Configuration Test email message is sent to the Host when they test the SMTP configuration. See "Testing Outgoing Email Settings" To locate this resource file, use the Find option on your web browser and search for EMAIL_SMTP_TEST_SUBJECT.Text.
