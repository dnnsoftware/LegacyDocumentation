User Lockout Email Message

The User Lockout email message is sent to the Administrator when a user attempts to login to a locked out user account. To locate this resource file, use the Find option on your web browser and search for EMAIL_USER_LOCKOUT_SUBJECT.Text and EMAIL_USER_LOCKOUT_BODY.Text

Here is the subject title and message for this email as provided in the default DNN core language pack:

[Portal:PortalName] User Account Lockout Alert

An attempt was made to access a locked out user account ( username: [Custom:0] ) on [Date:Current]. You may wish to investigate this matter further...

Thank you,

[Portal:PortalName]
