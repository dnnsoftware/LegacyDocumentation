Overview of the Type of Language Files

The Core Language Pack included with DNN includes language files which can be modified. Here's a list of the types of language files which can be edited using the Languages module.

  • Actions: The names given to common actions such as clear cache, delete module, edit module.
  • Strings: String resources are the names attributed Admin and Host pages, Banner types and Permission to Deploy modules.
  • Error Messages: Messages which appear when an error occurs.
  • Email Messages: Email messages are sent by DNN when users request password reminders, subscribe to a role, etc. They can also be message to editors or Administrators confirming actions such as deleting user accounts or sending newsletters.
  • Text: Text language files.
