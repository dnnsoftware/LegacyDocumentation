Adding One or More Pages

How to add one or more new pages to a site using the Pages module.

  1. Navigate to Admin > Page Management - OR - Go to a Pages modules.
  2. In the left-hand navigation tree, right-click on a page name and then select Add Page(s) from the drop down list.

  1. In the Pages multi-line text box, enter each page name on a separate line. If you wish to create a page hierarchy, simply add one right chevron characters ( > ) for each child level.
  2. Click the Create Page(s) button.

  1. The new pages are now added to the site menu, the Pages module navigation tree.
