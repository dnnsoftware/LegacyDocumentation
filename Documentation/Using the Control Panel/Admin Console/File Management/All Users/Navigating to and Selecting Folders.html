Navigating to and Selecting Folders

How to navigate to and select a folder using the Digital Asset Management Pro module. Any user (including anonymous users) who can view the Digital Asset Management Pro module select folders.

  • To view a subfolder: Click the Open button beside a folder to view its subfolders.
  • To hide a subfolder: Click the Close button beside a folder to close it and hide its subfolders.
  • To select a folder: Click on the folder name in the navigation tree. This highlight the selected folder name and displays the associated files in the Files Window. If you navigate away from DAM and then return to the page, the last folder that you selected will be displayed.
