Language Packs

  • See "Deploying Language Packs"

The following Language Packs are included in the default installation of DNN, one of which will have already been deployed upon installation:

  • Deutsch (Deutschland)
  • US English

  • Italiano (Italia)

Available Modules

See "Installing Optional Modules"

Evoq Content and Evoq Content Enterprise come packaged with the following optional modules: Announcements, SearchCrawlerSupport Package, Events, FAQs, Feedback, Form and List, IFrame, Links and Media. There are no optional modules included in the default installation of DNN, however they can be included prior to installation.
