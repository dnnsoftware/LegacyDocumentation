Managing Synonyms

How to create and manage a list of synonyms that can be used for searches. For example, if a users enters the word "Host" into the search box the site will return results for synonymous words such as "SuperUser".

Adding Synonyms

  1. Navigate to Admin > Advanced Settings >  Search Admin - OR - Go to a Search Admin module.
  2. Select the Synonyms tab.
  3. Click the Add Synonyms Group button to add a synonym to the list.

  1. In the Synonyms Tags text box, enter each synonymous word or phrase followed by a comma or by striking the Enter key.

  1. Click the Save button once all words are entered.

Deleting Synonyms

  1. Navigate to Admin > Advanced Settings >  Search Admin - OR - Go to a Search Admin module.
  2. Select the Synonyms tab.
  3. Click the Delete button beside the synonym group to be removed.

Editing Synonyms

  1. Navigate to Admin > Advanced Settings >  Search Admin - OR - Go to a Search Admin module.
  2. Select the Synonyms tab.
  3. Click the Edit button beside the required group.

  1. In the Synonyms Tags text box, add or remove synonymous words.

  1. Click the Save button to save your changes.
