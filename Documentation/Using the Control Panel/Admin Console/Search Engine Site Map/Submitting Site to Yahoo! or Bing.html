Submitting Site to Yahoo! or Bing

How to submit a site for indexing to either the Yahoo! or Bing search engine using the Sitemap module.

  1. Navigate to Admin > Advanced Settings > Search Engine Site Map - OR - Go to a Sitemap module.
  2. Expand the Site Submission section.
  3. At Search Engine, select either Bing or Yahoo!.
  4. Click the Submit link.
  5. Repeat Steps 3-4 to submit the site to the other search engine if desired.
