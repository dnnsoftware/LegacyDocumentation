Newsletter Error Messages

One of the following error messages will be displayed if a message fails to send.

  • "No Messages were sent because there are no recipients for this email." In this case, you must either select one or more roles at the User Role(s) field or enter one or more recipients into the Additional Emails field.
  • "No Messages were sent. A confirmation email has been sent to [From Email Address] with a description of the errors that happened during the sending process."
  • "An error occurred while sending the email: [Error Description]"
