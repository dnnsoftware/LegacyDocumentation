Delete a Module from Secondary Language

How to only delete a module a page that is available in a secondary culture so that the module is still available in the default language and any other cultures it has been added to.

  1. View the site in the required secondary language. See "Viewing a Site in a Secondary Culture"
  2. Navigate to the required page. See "Viewing Any Page" or See "Viewing any Page (Pages Module)"
  3. Delete the module from this page. See "Deleting a Module"
