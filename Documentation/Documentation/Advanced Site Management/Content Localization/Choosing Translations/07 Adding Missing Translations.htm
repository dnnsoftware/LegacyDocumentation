Adding Missing Language Translations to a Page

How to add each language translation that is missing back to a page. The translation could be missing if it had been deleted or if a new language was enabled after the initial translations were created. The translation that is added will be a new version based on the current content of the default language.

  1. Navigate to Admin > Advanced Settings > Languages - OR - Go to a Languages module.
  2. Select the Content Localization tab.
  3. At Select a Page, find or select the required page.
  4. Click the Add Missing Languages button located at the base of the module.

  1. Each missing translation is now added back to the page.
