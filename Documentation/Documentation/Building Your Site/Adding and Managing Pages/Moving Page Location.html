Moving Page Location (Iconbar)

How to move a page to a new location on the site menu using the Iconbar Control Panel.

  1. Go to the required page. See "Viewing Any Page"
  2. Go to the Page Functions section and then select  Settings.
  3. Go to the Basic Settings - Page Details section.
  4. At Parent Page, select a new parent page, or select < None Specified > to change this page to a parent page.
  5. Click the OK button to confirm.
