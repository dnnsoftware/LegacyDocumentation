Tagging Page Content

How to add a tag to a page. This functionality is only available on sites where tagging is included in page skins.

  1. Click the Add Tags link. This link is often located at the base of the page.
  2. Enter the tag name into the text box.
  3. Click the Save link. The newly added tag will now be listed at this field.

Tip: Tags can be managed using the Taxonomy Manager module. See "About the Taxonomy Manager Module"
