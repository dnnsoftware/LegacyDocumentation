Setting Paging Options

How to set the number of members records displayed on each page of the Member Directory and enable or disable the ability to view more results than displayed on the first page.
  2. Select the Member Directory Settings tab.
  3. Expand the Paging Optionssection.
  4. At Disable Paging, to disable the "Load More" button. This will only display the number of members displayed on the first page.
  5. At Page Size, select the number of members to be displayed on each page.

  1. Click the OK button to confirm.
