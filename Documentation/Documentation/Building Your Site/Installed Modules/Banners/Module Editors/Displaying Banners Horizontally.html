Displaying Banners Horizontally

How to set the Banners module to display banners horizontally across the page.

  1. Select Banner Options from the module actions menu.
  2. At Orientation, select Horizontal.
  3. Optional. In the Column Width text box, enter a number to set the width of each column cell. If this setting is left empty the banners may not align evenly across the module. E.g. Width is set at 300 pixels in the below image.
  4. Click the OK button to confirm.
