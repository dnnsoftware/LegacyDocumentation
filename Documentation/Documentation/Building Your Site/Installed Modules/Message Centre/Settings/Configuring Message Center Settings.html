Configuring Message Center Settings

If you choose to add the Message Center module to a page, the correct permissions must be configured for ensure it cannot be viewed by unauthenticated users. If permissions are not correct, the warning message "Module permissions are not properly set. The module shouldn't be visible for unauthenticated users. Please correct this." is displayed.

Here's how to set the correct permissions:

  2. Select the Page Settings tab.
  3. At Inherit View Permissions from Page, .
  4. In the View Module column, click on the Registered Users and (optionally) Unverified Users check boxes to grant permission to these roles.
  1. Click the OK button to confirm.
