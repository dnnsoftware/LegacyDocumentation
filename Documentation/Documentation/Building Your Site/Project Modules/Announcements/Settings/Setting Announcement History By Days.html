Setting Announcement History By Days

How to set the default number of days that each announcement will be published (displayed) on the Announcements module. Once the number of days set is reached the announcements are moved to the Expired group that can be viewed by choosing the "Select View" of Expired.

  2. Select the Announcements Settings tab.
  3. In the History (Days) text box, enter a number. E.g. 14 will display announcements with a publish date from 1 to 14 days ago. Announcements which are more than 14 days old will automatically expire.

  1. Click the OK button to confirm.
