Setting the Default View for Announcements

How to set the default view for announcements in the Announcements module. This setting enables users who are authorized to add a copy of an existing Announcements module on a different page and then set the default view of that module to expired, future, current or all, thereby allowing you to display an automatic archive of your announcements.

  2. Select the Announcements Settings tab.
  3. In the Default View text box, select one of the following from the drop down list:
    • Current: Current announcements are displayed by default.
    • Expired: Expired announcements are displayed by default.
    • Future: Future announcements are displayed by default.
    • All: All announcements are displayed by default.

  1. Click the OK button to confirm.
