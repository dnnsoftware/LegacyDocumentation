Viewing Event Details (Week View)

How to view the full details of an event when events are displayed in a weekly list. The event details page provides access to tools such as event enrollment and adding an event to your calendar.

  1. Click the Week View button and navigate to the event. See "Navigating to Events (Week View)"
  2. Click the [Event Title] link. Event details may be displayed when you hover your mouse over the event title.

  1. Click the Return link to return to the previous page.
