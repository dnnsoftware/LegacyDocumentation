Editing QueryString Parameters

How to edit QueryString Parameters on the IFrame module.

  1. Select Edit > Edit IFrame Options from the module actions menu to opens the Edit IFrame page.
  2. Maximize the QueryString Parameters section.
  3. Click the Edit button beside the parameter to be edited.

  1. Edit the Name and/or Value fields. See "Adding QueryString Parameters"
  2. Click the Save button.
  3. Repeat Steps 3-5 to edit additional QueryStrings.
  4. Click the Save button.
