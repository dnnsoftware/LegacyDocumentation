Deleting a QueryString Parameter

How to delete QueryString Parameters set on the IFrame module.

  1. Select Edit > Edit IFrame Options from the module actions menu to opens the Edit IFrame page.
  2. Maximize the QueryString Parameters section.
  3. Click the Delete button beside the parameter to be deleted. This displays the message "Are you sure you want to remove this item for the data source? Once remove it cannot be retrieved."
  4. Click the OK button to confirm deletion.

  1. Repeat Steps 3-4 to delete additional QueryStrings.
