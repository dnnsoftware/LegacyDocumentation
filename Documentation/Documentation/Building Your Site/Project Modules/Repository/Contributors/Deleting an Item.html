Deleting an Item

How to permanently delete an item from the Repository module. Authenticated users who are authorized to upload items are able to delete their own items. Unauthenticated users are unable to delete any items.

  1. Locate the item.
  2. Click the Edit   button beside the item.
  3. Click the Delete button. This displays the message "Are You Sure You Wish To Delete This Item?"
  4. Click the OK button to confirm.
