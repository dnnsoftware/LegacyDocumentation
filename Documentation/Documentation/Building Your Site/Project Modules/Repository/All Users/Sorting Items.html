Sorting Items

How to set the sort order of items displayed in the Repository module. Date is the default sort order. Sort order is not persistent if the page is refreshed or navigated away from.

  1. At Sort by, select the field you want to order items by. The following options are available: Date, Downloads, User Rating, Title, Author Name and CreatedDate.
  2. Optional. At Categories, select a category to filter items - OR - (if available) select All Items to view all items.
