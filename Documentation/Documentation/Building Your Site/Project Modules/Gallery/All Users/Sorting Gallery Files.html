Sorting Gallery Files

All site visitors can sort the files within albums in the Gallery module. The following options are typically available: Name, Size, Title, Author and Created Date. Additional fields which may be displayed are: Location, Score, OwnerID and Approved Date. Note: Sorting may be disabled.

  1. At Sort By, select the field to sort files by. The default setting is Title.
  2. Optional. At Desc, to sort albums in descending order.
