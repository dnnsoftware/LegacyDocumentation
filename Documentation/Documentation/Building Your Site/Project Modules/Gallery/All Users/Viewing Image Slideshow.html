Viewing Image Slideshow

All site visitors can view a slideshow of images inside an album of the Gallery module. Note: Slideshow may not be enabled.

  1. Navigate to the required album. See "Opening Albums"

  1. In Standard View/Card View, mouse over any image thumbnail and select  Slideshow - OR - In List View, click the Slideshow  button. This commences the slideshow from the chosen image. Images are displayed in accordance with the order in which they are sorted including descending order if enabled.
  2. Click the Back button when you have finished.
