Viewing Exif Metadata

All site visitors can view the Exif metadata of an image in the Gallery module. Exif (Exchangeable Image File Format) is a variation of the JPEG format and is used by most digital cameras. When you take a picture Exif Metadata information is stored for each picture. This information varies between cameras but can include information such as the date, time, camera settings, shutter speed, etc.

  1. Open the required album. See "Opening Albums"
  2. Select the EXIF Data button from the File Menu. See "Opening the File Menu"
