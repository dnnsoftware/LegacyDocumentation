Editing News Feeds

How to edit the news feeds displayed on the News Feeds (RSS) module.

  1. Select Edit > Edit Newsfeeds from the module actions menu.
  2. Click the Edit button beside the feed to be edited.

  1. Edit the details as required.
  2. Click the Update button.
  3. Repeat Steps 2-4 to edit additional feeds.
  4. Click the Return button to return to the page.
