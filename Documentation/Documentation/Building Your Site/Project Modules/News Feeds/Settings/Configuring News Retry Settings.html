Configuring News Retry Settings

How to configure the retry settings for retrieving news feeds on the News Feeds (RSS) module.

  2. Select the News Module Settings tab.
  3. In the Retry Times text box, enter the number of times the module should attempt to load a feed before waiting for a longer period. The default setting is 3.
  4. In the Retry TimeOut text box, enter the number of minutes between retries. The default setting is 120.
