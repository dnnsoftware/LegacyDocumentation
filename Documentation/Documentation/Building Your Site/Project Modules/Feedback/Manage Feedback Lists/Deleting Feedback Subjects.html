Deleting Feedback Subjects

How to permanently delete subjects from the Feedback module. As an alternative to deleting a feedback subject, you can deactivate it.

  1. Select Edit > Edit Category/Subject Lists from the module actions menu. This opens the Edit Lists page.
  2. At List Type, select Subjects.
  3. Click the Delete button beside the subject to be deleted. This displays the message "Are You Sure You Wish To Delete This Item?"
  4. Click the OK button to confirm.

  1. Repeat Steps 3-4 to delete additional subjects.
  2. Click the Return to Module button when you are finished.
