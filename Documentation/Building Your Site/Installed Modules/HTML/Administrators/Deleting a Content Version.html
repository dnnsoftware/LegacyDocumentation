Deleting a Content Version

How to delete a specific version of content using the HTML module.

  1. Select Edit Content from the HTML module actions menu.
  2. Expand the Version History section.
  3. Locate the required version.
  4. Optional. Click the Preview button to preview the version. This expands the Preview section and displays the selected version including details on the related workflow and the Item History.
  5. Click the Delete button. This displays a message similar to "Are you sure you wish to permanently delete version 3 [ that was created on [date] by [username]."

  1. Click the OK button to confirm.
