Exporting a Page

How to export a page which will be saved as a page template which can then be imported into any DNN site.

  1. Navigate to the required page. See "Viewing Any Page"
  2. On the ControlBar, hover over the Edit Pages menu option and then select Export Page

  1. At Folder, select a folder of the site's Digital Asset Management module (located on the Admin > File Management page) where the exported page will be stored.
  2. In the Template Name text box, the name of the exported page is displayed. You can either use this as the name of the page template or change it to a new name.
  3. In the Description text box, enter a description of the page you are exporting. This description is exported with the page template. Typically the description provides a brief overview of the page and (if included) it's content.
  4. Optional. At Include Content?, to include module content - OR - to add the modules without any content.
  5. Click the Export button.

  1. The path where the page template has been created is now displayed.
