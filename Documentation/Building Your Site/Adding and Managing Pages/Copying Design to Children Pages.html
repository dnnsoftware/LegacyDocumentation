Copying Design to Child Pages

How to copy the design applied to a page to all of its child (descendant) pages using the ControlBar. This applies the Page Skin and Page Container settings of the parent page to all child pages. This setting is only available for existing pages with child pages.

  1. Navigate to the parent page whose design you want to copy.
  2. On the ControlBar, hover over the Edit Page menu option and select Page Appearance.
  3. This will open the Page Settings popup on the Advanced Settings tab with the Appearance Section opened.
  4. Scroll down a little bit and click the Copy Design button. This displays the message "Skin and container will be replaced for all children of this page. Are you sure you want to continue?"

  1. Click the Yes button to confirm.
