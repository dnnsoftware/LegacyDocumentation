Editing a Wiki Page

How to edit the content of a Wiki page or add content to an existing page which has no content.

Important. To add content to the Home page of your Wiki, you must add a new page (See "Adding a Wiki Page") with the Page Name WikiHomePage. Once this has been done the first time, the Edit Wiki Topic link will be visible on the Home page.

  1. Navigate to or search for the required page. See "Navigating the Wiki" or See "Searching the Wiki"
  2. Click the Edit Wiki Topic link.

  1. In the Editor, add or edit page content.
  2. Optional. Click the Save & Continue link to save your changes to date and continue editing.
  3. Click the Save link to save and view the edited page.
