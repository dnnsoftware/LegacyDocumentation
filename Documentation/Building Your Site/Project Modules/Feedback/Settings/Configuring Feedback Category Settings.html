Configuring Feedback Category Settings

How to configure the option category settings for the Feedback module.

Tip: One or more categories must be created prior to configuring these settings. See "Adding Feedback Categories"

  2. Select the Feedback Settings tab.
  3. Expand the Feedback Category Settings section.
  4. At Category, select the category associated with this module. This displays the selected category name on the module. 
  5. At Category List Visible,  to enable users to select an alternative category - OR - to disable category selection.
  6. At Category Required,  if it is mandatory to select a category to submit feedback - OR - if selecting a category is optional.
  7. At Use Category Value as Send To,  to use the category value as the email address of the recipient of this message. Where no email address exists in the category value, then the "Send To" value is used - OR - to disable.

  1. Click the OK button to confirm.
