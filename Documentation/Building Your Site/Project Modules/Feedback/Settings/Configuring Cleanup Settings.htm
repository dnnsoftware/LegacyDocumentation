Configuring Cleanup Settings

How to automate the clean up of old comments from the Feedback Comments module.

  2. Select the Feedback Settings tab.
  3. At Pending , to clean up pending feedback, or to take no action for pending comments.
  4. At Private, to clean up private feedback, or to take no action for private comments.
  5. At Archived, to clean up archived feedback, or to take no action for archived comments.
  6. In the Days Before text box, enter the number of days history comments to be stored for before clean up. The default is 365 days
  7. In the Max Feedback text box, enter the maximum number comments to be stored for each feedback group (e.g. pending, private published and archived) before clean up. The default is 1000 days.
  8. Click the OK button to confirm.
