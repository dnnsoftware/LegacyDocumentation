Activating/Deactivating Feedback Categories

How to activate or deactivate a category on the Feedback module. Active categories are listed on the Settings page, whereas deactivated categories are not listed on the settings page and are only visible to Administrators on the Feedback Lists page.

  1. Select Edit > Edit Category/Subject Lists from the module actions menu. This opens the Edit Lists page.
  2. At List Type, select Categories.
  3. Click the Edit button beside the category to be activated or deactivated.
  4. At Is Active, to activate the category - OR - to deactivate it.
  5. At Is Portal Wide, if this category is available to all Feedback modules across this site. Note: This setting can only be checked using the module used to create the category - OR - if this category is currently inactive. Note: This setting cannot be unchecked if the category is in use on another module.
  6. Click the Save link.

  1. Repeat Steps 3-6 to modify additional categories.
  2. Click the Return to Module button when you are finished.
