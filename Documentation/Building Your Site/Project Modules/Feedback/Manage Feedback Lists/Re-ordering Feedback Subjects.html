Re-ordering Feedback Subjects

How to re-order subjects on the Feedback module. Feedback subjects are listed in the order in which they are created.

  1. Select Edit > Edit Category/Subject Lists from the module actions menu. This opens the Edit Lists page.
  2. At List Type, select Subjects.
  3. Click the Up  or Down  button beside a category to move it up or down one position in the list and then repeat as required.

  1. Repeat Steps 3 to re-order additional subjects.
  2. Click the Return to Module button when you are finished.
