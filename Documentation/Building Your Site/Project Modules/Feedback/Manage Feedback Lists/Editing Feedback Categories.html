Editing Feedback Categories

How to edit categories in the Feedback module.

  1. Select Edit > Edit Category/Subject Lists from the module actions menu. This opens the Edit Lists page.

  2. At List Type, select Categories.
  3. Click the Edit  button beside the category to be edited.

  1. Edit the category details as required.
  2. Click the Save link. The new details for this category are now displayed to the left.
  3. Repeat Steps 2-5 to edit additional categories.
  4. Click the Return to Module button when you are finished.
