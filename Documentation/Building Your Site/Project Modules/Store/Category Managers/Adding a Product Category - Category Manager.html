Adding a Product Category (Category Manager)

Category Managers and Catalog Managers can add a category to the Store using the Store Menu module.

Important. The SEO Feature option must be enabled to use the SEO Name and SEO Keywords fields. See "Setting Store Info"

  1. On the Store Menu module, select Edit  >   Add New Category from the Store Menu module actions menu. This opens the Edit Catalog page.

  1. Complete Steps 3 onwards in "Adding a Product Category"
