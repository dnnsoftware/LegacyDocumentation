Editing a Parent Category (Category Manager)

Category Managers and Catalog Managers can edit a parent category in the store using the Store Menu module. Note: Child categories can only be edited using the Store Admin module. See "Editing a Product Category"

  1. On the Store Menu module, click the Edit  button beside of the required parent category. This opens the Edit Catalog page.

  1. Edit one or more fields as required.
  2. Click the Update button.
