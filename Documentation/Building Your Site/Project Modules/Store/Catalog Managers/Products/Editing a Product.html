Editing a Product

Catalog Managers can edit a product in the Store using the Store Admin module.

  1. On the Store Admin module, click the Products link.
  2. At Category Filter, select the category associated with the required product.
  3. Click the Edit link beside the product.

  1. Edit the required fields. See "Adding a Product (Product Manager)" for details of available fields.
  2. Click the Update button.
