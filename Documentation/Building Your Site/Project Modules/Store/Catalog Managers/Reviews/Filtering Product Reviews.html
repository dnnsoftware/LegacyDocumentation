Filtering Product Reviews

Catalog Managers can filter product reviews using the Store Admin module.

  1. On the Store Admin module, click the Reviews link and then apply one or more of these filters.
  2. At Status, select from these options:
    • --- ALL ---: Select to view both approved and unapproved reviews.
    • Not Approved: Select to only view unapproved reviews. These reviews are not visible to customers.
    • Approved Only: Select to view only reviews that are approved. These reviews are visible to customers.
  3. At Category, select --- ALL --- to view reviews for products in all categories - OR - Select a category to only view reviews for products within that category.
  4. At Product, select --- ALL --- to view reviews for all products - OR - Select a product to only view reviews for that product.
