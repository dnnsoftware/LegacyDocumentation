Canceling an Order

How to cancel an order placed in the Store using the Store Account module. This option may not be available on all stores.

Prerequisite. Order cancellation must be authorized. See "Setting Store Info"

Option One:

  1. Go to the Store Account module.
  2. Click the Order History link.
  3. Click the Details link beside the required order.

  1. Click the Cancel Order button. The order status is updated to "Canceled".


Option Two:

  1. Go to the Store Account module.
  2. Click the Order History link.
  3. Click the Cancel button beside the required order.
