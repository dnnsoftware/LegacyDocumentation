Deleting a Shipping Rate

How to permanently delete one or more shipping rates from the Store module.

  1. Go to the Store Admin module.
  2. If the Store Info page isn't displayed, click the Store Info link.
  3. Maximize  theShipping Provider Settings section. 
  4. In the Delete column, beside each shipping rate to be deleted.


  1. Click the Update Shipping Settings link.
  2. Repeat Steps 4-5 to delete additional rates.
