Enabling/Disabling Authorize.Net Test Mode

How to enable or disable the Authorize.Net test mode on the Store. Enabling Test mode allows orders to be sent to the payment gateway without taking live transactions. This tutorial assumes Authorize.Net has already been configured as the gateway. See "Configuring Authorize.Net as the Payment Gateway"

  1. Go to the Store Admin module.
  2. If the Store Info page isn't displayed, click the Store Info link.
  3. At Test Mode, select from these options:
    • Mark  the check box to use the store in test mode only (Transactions are not live in this mode).
    • Unmark  the check box to begin accepting payments.

  1. Click the Update button.
