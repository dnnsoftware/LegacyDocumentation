Enabling Manual Document Sorting

How to enable module editors to manually set the sorting of documents displayed in the Documents module. Enabling this setting will display the "Sort Index" text box on the Edit Documents page, allowing Module Editors to manually enter a number to set the order that documents appear in the module.

Tip: Manual Sorting is the default setting for newly added modules.
