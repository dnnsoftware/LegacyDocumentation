Re-ordering List Records

How to re-order list records by column on the Form and List module. Note: This option may not be available for some lists.

  1. Click on a [Column Name] to order list records by that column. For Example, in the below image, the Up arrow  is displayed beside the "Bike Name" column indicating that it is sorted in ascending order.
  2. Click the [Column Name] a second time to reverse the order. In this example this would reorder the records in descending order and display the Down arrow beside the "Bike Name" column.
