Editing List or Form Columns

How to edit one or more columns in a list or form created using the Form and List module.

Warning. Do not modify the Type field of any column once data has been entered into a list as it may cause an error.

  1. Select  Form and List Configuration from the module actions menu.
  2. Click the Edit button beside the column to be edited. This enables editing of the column including any advanced column options.

  1. Edit columns as required.
  2. Click the Save button.
  3. Repeat Steps 2-4 to edit additional columns.
  4. Click the Save Configuration and Return button to return to the module.
