Settings for User Profile Links

How to configure the optional setting for user profile links in a list on the Form and List module.

  1. Select  Form and List Configuration from the Form and List module actions menu - OR - Click the Form and List Configuration  button.
  2. Maximize  the Schema Definition section.
  3. Maximize the More Column Settings section.
  4. At Settings for User Profile Links set the following options:
    1. At Open in New Window, to open the profile in a new Web browser window - OR - to open the profile in the same window.
    2. At User Link Caption select from the following options to set which name to display as the link to User Profile:
      • Display Name
      • User Name
    3. At Add to CreatedBy/UpdatedBy?, to add a link to the user's profile on both the Created By and Updated By columns - OR - for no link.
  5. Click the Save Configuration and Return link.
