Advanced Look-Up Column Options

How to configure the optional Advanced Column Options for look-up columns for a form or list in the Form and List module. This tutorial assumes you are currently adding or editing a Look-Up column. See "Creating a List", "Creating a Form" or "Editing List or Form Columns"

  1. Go to the Advanced Column Options section and set any of the following optional fields:
  2. At Source Column, enter the name of the column containing a ref to a user or local file.
  3. In the Token Value text box, enter the token text to query the User or File object.
  4. At Show on Edit Form?, if this value will be displayed to all users when editing - OR -  to hide the field on the Edit Form.

Tip: In the below example, the Look-Up column is set to display the size of the file selected in the Brochure column.
