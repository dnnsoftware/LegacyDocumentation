Adding a Separator Field

How to add a separator field to a form or list using the Form and List module. This column type displays a separator (line break) on the form or list which is useful for grouping similar fields and providing visual cues to users. This tutorial assumes you are currently adding a separator column to a form or list (See "Creating a Form or List") or editing the settings of an existing separator field. See "Editing List or Form Columns"

  1. In the Title text box, enter a title that will be displayed above the separator.
  2. At Required, to set the field as mandatory - OR - to set the field as optional. This is the default setting.
  3.  At Display on List, if this column is visible on the list. This is the default setting. - OR - to set the field as visible to Administrators only.
  4.  At Restricted Form Field, if this column is visible to all users - OR - if the column is only visible to users/roles for Restricted fields as set in permissions. This is the default setting. See "Setting Form Permissions"
  5. Go to the List Settings section.
  6.  At Collapsible Area, if the fields belonging this separator are collapsible. This creates a collapse/expand section that continues to either the next Separator or the end of the form or list - OR - for no collapsible area. This is the default setting.
  7. Click the Save button.
