Viewing Enrolled Users

How to view details of users who are enrolled for an event on the Events module. The Enrolled Users list displays the Display Name, email address and number of attendees enrolled against that user's account, as well as whether the enrollment has been approved. Editors can click on the Display Name to view the enrollee's user profile, or click on their email address to send them a message. Note: This option may not be available to all users.

  1. Go to the event details page of the required event. See "Viewing Event Details (List View)", See "Viewing Event Details (Month View)" or See "Viewing Event Details (Week View)".
  2. Optional. The Enrolled Users table may be displayed on this page depending on the module settings.

  1. Click on the Edit link to view the event details and the Enrolled Users list.
