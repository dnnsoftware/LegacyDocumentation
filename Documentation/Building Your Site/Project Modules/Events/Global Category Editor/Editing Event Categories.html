Editing Event Categories

How to edit one or more categories on the Events module.

  1. Select Edit Categories from the Events module actions menu.
  2. In the Category Name list, click on the linked [Category Name]. This displays the category details in the Category and Color fields.
  3. Edit fields as required. See "Adding Event Categories"
  4. Click the OK button to confirm.

  1. Repeat Steps 2-4 to edit additional categories.
  2. Click the Return link to return to the module.
