Viewing Info Link

How to view the description of a link within the Links module. This optional setting may not be enabled on all or some links. If a description wasn't added for the link the info link will still be displayed however no details are displayed when it is clicked.

  1. Click the info link ( ... ) displayed to the right of the link to hide or show the link description.
