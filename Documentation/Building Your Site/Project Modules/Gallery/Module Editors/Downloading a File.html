Downloading a File

Module Editors can download a file from the Gallery module. Note: You must be in one of the download roles (which may include "All Users") specified in the Gallery configuration to download files.

  1. Open the required album. See "Opening Albums"
  2. Locate the required file. Then perform one of the following:
    • In List View, select then Download This File button.

  • In Standard View and Card View, mouse over file and then select Download.

Tip: For image file types, the file to be downloaded will be the full-resolution file which was originally uploaded rather than a reduced resolution file such as that presented when viewing the image.
