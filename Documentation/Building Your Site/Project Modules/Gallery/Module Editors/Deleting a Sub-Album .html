Deleting a Sub-Album

Module Editors can delete a sub-album (child album) from the Gallery module.

  1. Navigate to the parent album of the sub-album to be deleted and open menu. See "Opening the Album Menu"
  2. Select Edit This Album from the album menu.

  1. Go to the Sub-Albums and Files Currently Contained In This Album section.
  2. Click the Delete button beside the album to be deleted. This displays the message "Are You Sure You Want To Delete The Album Named '[Album Name]'?

  1. Click the OK button to confirm.
  2. Repeat Steps 4-5 to delete additional albums.
