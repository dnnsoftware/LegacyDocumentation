Adding News Feeds from Syndicated Modules

How to add news feeds from syndicated modules on your website using the News Feeds (RSS) module.

Prerequisite. Syndication must be enabled for the required module. See "Configuring Basic Page Settings for Modules"

  1. Go to a syndicated module. In this example an Announcements module is used.
  2. Select Manage > Syndicate from the module actions menu.

  1. This displays the XML code for the module content in a new Web browser. Note: You may need to allow pop-ups to view the code.
  2. Copy the URL that is displayed in the Address bar of the Web browser and paste it to your clipboard.

  1. Click the Back button on the Web browser to return to the site and then go to a News Feed (RSS) module.
  2. Select Edit > Edit Newsfeeds from the module actions menu.
  3. Click the Add Feed button.
  4. In the Feed Url text box, enter/paste the copied URL.

  1. In the Cache Time text box, enter the number of minutes between refreshes of this feed - OR - Leave this field as -1 to use the default setting for this feed.
  2. Click the Update button. This displays the name of the newly added feed.

  1. Optional. Repeat Steps 7-10 to add additional feeds.
  2. Click the Return button to return to the module.
