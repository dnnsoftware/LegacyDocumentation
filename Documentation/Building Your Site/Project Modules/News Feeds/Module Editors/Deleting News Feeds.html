Deleting News Feeds

How to delete one or more news feeds from the News Feeds (RSS) module.

  1. Select Edit > Edit Newsfeeds from the module actions menu..
  2. Click the Edit button beside the feed to be deleted.
  3. Click the Delete button at the base of the page. This displays the message "Do you really want to delete the selected RSS feed?"
  4. Click the OK button to confirm.
  5. Repeat Steps 2-4 to delete additional feeds.
  6. Click the Return button to return to the page.
