Deleting a Module

How to delete a module from a page.

Tip: Deleted modules are stored in the Recycle Bin.

  1. Mouse over the Manage button on the module action menu.
  2. Select Delete from drop down list. This displays the message "Are You Sure You Wish To Delete This Module?"

  1. Click the OK button to confirm.
