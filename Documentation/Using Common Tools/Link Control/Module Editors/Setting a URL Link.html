Setting a URL Link

How to set a link to any URL using the Link Control. The URL can be to any resource or file available on the Internet including website domains, web pages, files, or images.

  1. At Link Type, select URL ( A Link To An External Resource ).
  2. In the Location ( Enter The Address Of The Link ) text box, enter the URL. E.g. http://www.domain.com.
