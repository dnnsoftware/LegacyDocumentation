About the Editor

The Editor provides basic and rich text editing capabilities across numerous modules including Announcements, FAQ, Events and HTML/HTML Pro. The Basic Text Box associated with the Editor is the same for all DNN versions. The Rich Text Editor (RTE) associated with the Editor provider will depend upon the DNN version and edition of you are using.

This manual details how to use two different providers for the RadEditor control, the DotNetNuke.RadEditorProvider (default) and the TelerikEditorProvider, which are both included with DNN 6+.

Other RTE's which can be integrated with DNN include the FCKEditor and the Free Text Box.
