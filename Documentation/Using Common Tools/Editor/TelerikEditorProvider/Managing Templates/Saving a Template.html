Saving a Template

How to create and save a template using the TelerikEditorProvider for the RTE.

  1. Create your template by adding content such as text, images and tables as desired.

  1. Click the Save Template button. This opens the Save As Template window.
  2. At Folder, select the folder where the template will be saved to.
  3. In the File Name text box, enter a name for this template.
  4. At Overwrite If File Exists?, to overwrite any template that exists with this template - OR - if you don't want to override an existing file. This enables warning message if a template with this name already exists.
  5. Click the Save button.
    • If there is no conflict with overwriting, the message "The template was saved successfully" is displayed. Click OK to close the window.
    • If there is a conflict, a message reading "The template was not saved. The following error was reported: The file already exists" is displayed. In this case, repeat Steps 2-6, entering a different file name.
