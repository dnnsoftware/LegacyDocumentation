Paragraph Toolbar

The Paragraph toolbar of the TelerikEditorProvider RTE has the following tools:

Tool Icon Description
SuperScript Transforms selected content to superscript.
Subscript Transforms selected content to subscript.
New Paragraph Inserts a new paragraph.
Insert Groupbox Inserts a group box. Use to layout forms and to label sections of your content.
Horizontal Rule Inserts a horizontal line.
Insert Date Inserts the current date according to your computer.
Insert Time Inserts the current time according to your computer.
Format Code Block Enables you to insert and format a block of code and have it render as text. See "Formatting Code Block"
