Formatting Toolbar

The Formatting toolbar of the TelerikEditorProvider RTE has the following tools:

Tool Icon Description
Text Formatting
Bold (CTRL + B) Add/remove bolding to selected content.
Italicize (CTRL + I) Add/remove italics to selected content.
Underline (CTRL + U) Add/remove underline to selected content.
Strikethrough Add/remove strikethrough to selected content.
Alignment
Align Left Left align the selected content.
Align Center Center align the selected content.
Align Right Right align the selected content.
Justify Justify align the selected content.
Remove Alignment Remove alignment formatting from selected content.
Indent and Lists
Indent Indent selected content.
Outdent Outdent selected content.
Numbered List Create a numbered list.
Bullet List Create a bullet list.
Other
Show/Hide Borders Click to show or hide all table borders. This is useful where table borders are not displayed.
XHTML Validator Click to open the XHTML Validator window. At the very top of the window, select the radio button to set the type of XHTML validation you wish to perform. This displays the validation results and details.
