Troubleshooting: Image Not Displaying

Some images may not display in the Editor.

This can occur if the image is set as hidden.

  1. Unmark  the Hidden attribute check box at the for the image. See "Managing File Properties" 
  2. Return to the module to see if the image is displaying. You may need to Refresh (Hold down the Shift key and strike the F5 key) the page to see the changes.

This can also occur if module caching is set for too longer period. In this case, extend the caching time as required. See "Configuring Cache Settings for a Module"
