Using the Table Wizard

How to design a table using the Table Wizard of the TelerikEditorProvider for the RTE.

  1. Click the Insert Table button.
  2. Select Table Wizard. This opens the Table Wizard on the Table Design tab. A table of two columns by two rows is displayed as the basis for your design.
  3. Optional. Select a cell. This enables the Increase and Decrease buttons which are available to change the design.
  4. Select an Increase or Decrease button to modify the table design. You can choose to add columns and rows, as well as span columns and rows.
  5. Optional. Change tabs to set other properties.
  6. Click the OK button.
