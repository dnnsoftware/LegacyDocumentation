Pasting HTML Content

How to paste HTML content into the Editor control.

  1. Select the Basic Text Box radio button located above the Editor. This displays the Editor as a Basic Text Box. The content within the Editor is displayed as HTML.
  2. At Render Mode (located below the Editor), select Html.
  3. Paste in your Html.

Tip: HTML can also be pasted into the Raw Render Mode
