Editing an Email or URL Link

How to edit a link in the DotNetNuke.RadEditorProvider for the RTE.

Option One:

  1. Select the linked text or object.
  2. Click the Hyperlink Manager (CTRL + K) button. This opens the Hyperlink Manager.
  3. Edit the link as required. For more details on the available fields, See "Adding an Email Link" or See "Adding a URL Link"
  4. Click the OK button.

Option Two:

  1. Right-click on the linked text or object.
  2. Select Properties... from the drop down list. This opens the Hyperlink Manager.
  3. Edit the link as required. For more details on the available fields, See "Adding an Email Link" or See "Adding a URL Link"
  4. Click the OK button.
