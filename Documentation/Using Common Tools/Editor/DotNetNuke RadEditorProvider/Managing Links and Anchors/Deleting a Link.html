Deleting a Link

How to remove a link from the DotNetNuke.RadEditorProvider for the RTE.

Option One:

  1. Select the linked text or object.
  2. Click the Remove Link (CTRL + SHIFT + K) button.

Option Two:

  1. Right-click on the linked text or object.
  2. Select Remove Link from the drop down list. This opens the Hyperlink Manager.
