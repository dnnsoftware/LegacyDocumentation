Setting Table Accessibility

How to set the accessibility of a new or existing table using the Table Wizard of the DotNetNuke.RadEditorProvider for the RTE. The below tutorial demonstrates how to set one heading row and add a caption to the table.

  1. Right-click inside a cell of an existing table and select Table Properties - OR - Open the Table Wizard. See "Inserting a Table"

  1. Select the Accessibility tab and set any of these Accessibility Options:
    1. In the Heading Rows text box, enter the number of rows which are headings. The maximum number of heading rows for the current table is displayed to the right of this field. E.g. 1
    2. In the Heading Columns text box, enter the number of columns which are headings. The maximum number of heading columns for the current table is displayed to the right of this field.
    3. In the Caption text box, enter a caption to be displayed above the table.
    4. row associated with the Alignment Selector button and then select the alignment of the caption. If no alignment is selected, the default is center alignment.
    5. In the Summary text box, enter a summary of the table contents. The table summary isn't displayed on the page, but can be read using accessibility tools such as text readers.
    6. At Associate Cells With Headers, to associate cells with headers - OR - to disable.
    7. Optional. Select a new tab to set additional properties.

  1. Click the OK button confirm.
